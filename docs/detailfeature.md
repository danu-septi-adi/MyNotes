# Detail Feature — Blueprint Final

> Status: **FIX**
> Expo SDK 52+ | Local Database (SQLite)

---

## 1. Catatan Keuangan

**Model Transaksi:** jumlah, kategori, tanggal, tipe (income/expense), catatan ✅

**Budget:** Budget per kategori per bulan + total budget bulanan

**Grafik:** Pie chart (kategori) + Bar chart (per hari/minggu/bulan)

**Hutang/Piutang:** Tracking dengan tanggal jatuh tempo

---

## 2. Catatan Aktifitas Harian

**Status:** Belum, Sedang, Selesai, Batal
**Prioritas:** High, Medium, Low

**Alarm:** Push notification + notifikasi in-app
**Journal:** Satu halaman dengan tab (Todo + Journal)

---

## 3. Wishlist

**Field:** Nama barang, harga, prioritas (Mendesak/Nanti/Nabung dulu)
**Link Tabungan:** Input nominal target + progress otomatis dari catatan keuangan + manual update

---

## 4. Database & Data

**Export:** CSV + JSON
**Import:** CSV + JSON

---

## 5. Navigasi & UI/UX

**Bottom Nav:** 5 menu — Home, Finance (investing masuk sini), Aktifitas, Wishlist, Profil
**Dashboard Widget:** Saldo, Pengeluaran hari ini, Todo hari ini
**Trading:** Masuk di menu Profil/Lainnya

---

## 6. Jurnal Trading

**Field:** pair, volume, harga entry, SL, TP, alasan entry, hasil
**Ringkasan P/L:** Per trading + per pair + per bulan

---

## 7. Jurnal Investing

**Real-time API:** CoinGecko (crypto) + GoldAPI (emas) + Alpha Vantage (saham)
**Field:** Nama aset, jumlah unit, harga beli, total invest, catatan

---
